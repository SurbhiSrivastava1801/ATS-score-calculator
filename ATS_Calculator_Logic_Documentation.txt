- Standard Sections: Must include 'experience', 'education', 'skills' (20 pts)
- ATS-Friendly: No special chars like {}, [], symbols (20 pts)

5. SCORE CALCULATION PROCESS
=====================================================

┌─────────────────────────────────────────────────────────────────┐
│                    SCORE CALCULATION                           │
├─────────────────────────────────────────────────────────────────┤

┌─────────────┐ ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ Formatting  │ │ Keywords    │ │ Contact     │ │ Experience  │
│    Score    │ │   Score     │ │   Score     │ │   Score     │
│     85      │ │     92      │ │     100     │ │     78      │
└─────────────┘ └─────────────┘ └─────────────┘ └─────────────┘

┌─────────────┐ ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ Education   │ │ Skills      │ │ Length      │ │ Achieve     │
│   Score     │ │   Score     │ │   Score     │ │   Score     │
│     95      │ │     88      │ │     100     │ │     82      │
└─────────────┘ └─────────────┘ └─────────────┘ └─────────────┘

        │         │         │         │         │         │         │         │
        ▼         ▼         ▼         ▼         ▼         ▼         ▼         ▼
┌─────────────────────────────────────────────────────────────────┐
│  OVERALL SCORE = (85+92+100+78+95+88+100+82) ÷ 8 = 90%        │
└─────────────────────────────────────────────────────────────────┘

CALCULATION FORMULA:
Overall Score = (Score1 + Score2 + Score3 + Score4 + Score5 + Score6 + Score7 + Score8) ÷ 8

Each criterion contributes equally to the final score (12.5% weight each).

11. KEY LOGIC SUMMARY
=====================================================

CORE PRINCIPLES:
1. TRANSPARENCY: Every score is explained with specific issues and suggestions
2. COMPREHENSIVE: Covers all major ATS evaluation criteria
3. ACTIONABLE: Provides specific, implementable recommendations
4. INDUSTRY-AWARE: Analyzes keywords across multiple industries
5. USER-FRIENDLY: Clear visual feedback and downloadable reports
6. ROBUST: Handles various file types and error conditions gracefully

SCORING METHODOLOGY:
- 8 equally weighted criteria (12.5% each)
- Each criterion scored 0-100 points
- Overall score = average of all individual scores
- Transparent calculation process displayed to user

ANALYSIS DEPTH:
- Formatting: Structure, headers, bullet points, special characters
- Keywords: 13 common resume keywords
- Contact: Email, phone, location, professional links
- Experience: Section, job titles, companies, dates
- Education: Section, degree, institution
- Skills: Section, technical skills, soft skills
- Length: Word count optimization (300-800 words ideal)
- Achievements: Quantified results, action verbs, impact keywords

INDUSTRY ANALYSIS:
- 5 industry categories with 12-22 keywords each
- Percentage match calculation for each industry
- Missing keyword identification
- Industry-specific recommendations

RECOMMENDATION SYSTEM:
- Score-based priority levels (High/Medium/Low)
- Category-specific improvement suggestions
- Actionable, specific advice
- Always includes general optimization tips
